# Ubuntu based Python mongo client,
## with Azure blob storage, and AWS CLI.


### docker image: 
    `murvudd/python-mongo:latest`
    
    https://hub.docker.com/repository/docker/murvudd/python-mongo

### build:

```bash
make
```

### deploy:

```bash
make release
```
